# Task 4: Sentiment Analysis - Amazon Product Reviews

## What I Did
Analyzed 20 Amazon product reviews to classify them as Positive, Negative, or Neutral.

## Tools Used
- Python
- TextBlob (sentiment analysis library)
- Pandas
- Matplotlib

## Results
| Sentiment | Count | Percentage |
|-----------|-------|------------|
| Positive  | 10    | 50%        |
| Negative  | 5     | 25%        |
| Neutral   | 5     | 25%        |

## Key Insights
- 50% of customers are happy with their purchases
- 25% expressed dissatisfaction (quality issues)
- Recommended improvements: quality control, return process
